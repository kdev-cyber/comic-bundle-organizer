\# Comic Bundle Organizer



A Python automation tool that organizes digital comic bundles into a clean folder structure.



\## Features



\- Extracts ZIP files automatically

\- Detects comic issues by filename

\- Organizes files into categories:

&#x20; - Issues

&#x20; - Covers

&#x20; - Manga

&#x20; - Extras

&#x20; - Photos

&#x20; - Videos

&#x20; - Scripts

&#x20; - Review

\- Prevents file overwrites

\- Cleans up empty folders



\## How It Works



1\. Place your comic bundle files into a folder

2\. Run the script

3\. Enter:

&#x20;  - Folder path

&#x20;  - Series name

&#x20;  - Optional short code

4\. Files are automatically sorted



\## Supported File Types



\- Comics: PDF, CBZ, CBR

\- Images: JPG, PNG, WEBP, GIF, TIF

\- Videos: MP4, MOV, AVI

\- Scripts: DOCX, DOC, TXT



\## Folder Structure



Issues  

Covers  

Manga  

Extras  

Photos  

Videos  

Scripts  

Review  



\## Notes



\- Unknown or unmatched files go into the Review folder

\- Script is designed for Kickstarter-style comic bundles
